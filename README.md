# 介绍
使用 solr 作为检索服务，实现了全文检索功能，仿制了一个京东商城，实现了“京东”的搜索功能。

1. 支持搜索后的类别筛选
2. 支持价格区间过滤
3. 支持按照价格排序
4. 实现了分页功能

# 环境
1. jdk 1.8
2. maven
3. solr 8.2
